# Student Task Tracker Go Web App

## Project Description

Student Task Tracker is a simple web application that helps students manage their school tasks. Students can add tasks such as homework, projects, or study plans. They can view all their tasks in one list so they can easily remember what they need to do. When a task is finished, they can mark it as completed. They can also delete tasks that are no longer needed. This app helps students stay organized and manage their time better.

## Expected Features

- Add task
- View task list
- Mark task as completed
- Delete task
